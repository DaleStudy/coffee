# 매칭은 어떻게 이루어지나요?

"왜 이 사람이랑 매칭됐지?" 궁금하셨다면, 여기서 그 비밀을 알려드릴게요.

## 매칭의 3가지 원칙

### 1. 같은 사람과 또 만나지 않아요

최근 매칭 이력을 확인해서 같은 조합이 반복되지 않도록 합니다. 덕분에 매번 새로운 사람을 만날 수 있어요.

### 2. 적게 만난 사람을 우선 매칭해요

모든 참여자 조합에 **점수**를 매깁니다. 아직 만나지 않았거나, 만남 횟수가 적은 사람과의 점수가 높아요.

| 만남 횟수 | 점수 | 의미 |
|----------|------|------|
| 0회 | ★★★★★ | 최우선 매칭 |
| 1회 | ★★★☆☆ | 적당히 우선 |
| 2회 | ★★☆☆☆ | 가능하지만 후순위 |
| 3회+ | ★☆☆☆☆ | 다른 사람 먼저 |

### 3. 새 멤버와 오래된 멤버를 섞어요

참여 횟수에 따라 멤버를 세 그룹으로 나눕니다:

- **뉴비**: 참여 횟수 하위 25%
- **레귤러**: 중간 50%
- **베테랑**: 참여 횟수 상위 25%

그리고 **서로 다른 그룹끼리** 만날 확률을 높입니다.

| 조합 | 매칭 선호도 |
|------|-----------|
| 뉴비 + 베테랑 | 🔥 매우 높음 |
| 뉴비 + 레귤러 | 👍 높음 |
| 레귤러 + 베테랑 | 👍 높음 |
| 레귤러 + 레귤러 | 🤝 보통 |
| 베테랑 + 베테랑 | 😌 낮음 |
| 뉴비 + 뉴비 | 😌 낮음 |

새로 참여한 분이 커뮤니티의 경험 많은 멤버를 만나면 적응이 훨씬 빨라지거든요!

## 최종 점수는 이렇게 계산돼요

```
최종 점수 = 만남 횟수 점수 (60%) + 경험 믹싱 점수 (40%)
```

만남 횟수가 더 큰 비중을 차지합니다. 다양한 사람을 만나는 것이 가장 중요한 가치이기 때문이에요.

## 홀수일 때는?

참여자가 홀수이면 한 조만 3인조가 됩니다. 3인조도 색다른 매력이 있으니 즐겨보세요!

## 한마디로 정리하면

> 봇이 알아서 **가장 다양하고, 균형 잡힌 매칭**을 만들어줍니다. 여러분은 매칭 결과를 확인하고 대화를 즐기기만 하면 돼요!
