# 커피챗 참여 & 중단 방법

## 커피챗에 참여하기

디스코드 서버의 아무 채널에서 슬래시 명령어를 입력하면 됩니다.

```
/coffee join
```

명령어를 입력하면 봇이 바로 확인 메시지를 보내줍니다. 이후 다음 매칭부터 자동으로 포함됩니다.

> 참여 후 별도로 해야 할 일은 없습니다. 매칭 결과가 나올 때까지 기다리면 돼요!

## 커피챗 중단하기

일정이 바쁘거나 쉬고 싶을 때는 언제든 탈퇴할 수 있습니다.

```
/coffee leave
```

탈퇴해도 다시 `/coffee join`으로 언제든 복귀할 수 있어요. 부담 없이 참여하세요.

## 자주 묻는 질문

**Q. 참여 후 바로 매칭되나요?**

아니요. 다음 매칭 일정에 포함됩니다. 매칭은 격주로 진행되며, 정확한 일정은 [스케줄 안내](/docs/schedule)를 확인하세요.

**Q. 여러 그룹에 동시에 참여할 수 있나요?**

네! 서버에 여러 커피챗 그룹이 있다면, 각 그룹의 역할(Role)을 받아 동시에 참여할 수 있습니다.

**Q. 탈퇴하면 매칭 이력이 삭제되나요?**

아니요. 이력은 그대로 유지됩니다. 복귀 시 이전에 만났던 사람과 중복 매칭되지 않도록 이력이 활용돼요.
