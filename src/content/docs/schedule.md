# 매칭 스케줄

## 매칭은 언제 진행되나요?

커피챗 매칭은 **격주 월요일 오전 9시(KST)**에 자동으로 진행됩니다.

| 항목 | 내용 |
|------|------|
| 매칭 주기 | 2주마다 |
| 매칭 요일 | 월요일 |
| 매칭 시간 | 오전 9:00 (한국 시간) |
| 결과 공지 | 매칭 직후 디스코드 채널에 자동 공지 |

## 매칭 결과는 어디서 확인하나요?

매칭이 완료되면 지정된 디스코드 채널에 봇이 결과를 자동으로 공지합니다. 각 조별로 쓰레드가 생성되어 바로 대화를 시작할 수 있어요.

## 매칭 후 커피챗은 언제까지 하면 되나요?

다음 매칭 전까지 진행하면 됩니다. 보통 **2주 안에** 편한 시간을 잡아 대화하면 돼요. 짧게 15분도 좋고, 길게 1시간도 좋습니다.

## 매칭에서 빠지고 싶은 주가 있다면?

특정 기간 동안 매칭을 원하지 않으면 `/coffee leave`로 잠시 탈퇴한 뒤, 다시 참여하고 싶을 때 `/coffee join`을 입력하세요.

## 그룹별 스케줄이 다를 수 있나요?

네. 서버에 여러 커피챗 그룹이 있다면, 그룹마다 매칭 주기나 그룹 크기가 다를 수 있습니다. 그룹별 상세 설정은 서버 운영자에게 문의하세요.
